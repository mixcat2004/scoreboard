# 四人計分板 PWA — 安裝說明

## 你有哪五個檔案

```
📁 計分板/
├── index.html      ← 主程式
├── manifest.json   ← App 設定
├── sw.js           ← 離線支援
├── icon-192.png    ← App 圖示（小）
└── icon-512.png    ← App 圖示（大）
```

---

## 第一步：免費上傳到 GitHub Pages

### 1-1 註冊 GitHub（已有帳號可跳過）
前往 https://github.com → 點「Sign up」免費註冊

### 1-2 建立新 Repository
1. 登入後點右上角「+」→「New repository」
2. Repository name 填：`scoreboard`
3. 選「**Public**」
4. 點「**Create repository**」

### 1-3 上傳五個檔案
1. 進入剛建立的 repo
2. 點「**uploading an existing file**」
3. 把五個檔案**全部拖曳**進去
4. 點「**Commit changes**」

### 1-4 開啟 GitHub Pages
1. 進入 repo → 點上方「**Settings**」
2. 左側選「**Pages**」
3. Source 選「**Deploy from a branch**」
4. Branch 選「**main**」，資料夾選「**/ (root)**」
5. 點「**Save**」
6. 等約 1 分鐘，頁面上方會出現你的網址：
   ```
   https://你的帳號.github.io/scoreboard/
   ```

---

## 第二步：安裝到 Android 手機

1. 用 **Chrome** 開啟你的網址
2. 點右上角「⋮」選單
3. 點「**安裝應用程式**」或「**加入主畫面**」
4. 點「安裝」確認

✅ 桌面出現「計分板」圖示，點開就像 App！

---

## 第三步：安裝到 iPhone

1. 用 **Safari**（必須用 Safari）開啟你的網址
2. 點底部中間的「**分享**」按鈕 ⎋
3. 往下滑找「**加入主畫面**」
4. 名稱可改成「計分板」→ 點「新增」

✅ 桌面出現「計分板」圖示，點開就像 App！

## 常見問題

**Q: iPhone 安裝後語音無法使用？**
A: iOS Safari 語音需要 HTTPS，GitHub Pages 預設就是 HTTPS，應正常運作。若仍無效，請到「設定 → Safari → 麥克風」確認已開啟權限。

**Q: Android 沒出現「安裝」選項？**
A: 確認使用 Chrome 瀏覽器，且網站已透過 HTTPS 開啟（網址開頭是 https://）。

**Q: 分數會不見嗎？**
A: 分數儲存在手機本地（localStorage），清除瀏覽器資料才會消失。建議定期用「匯出 CSV」備份。

**Q: 朋友可以一起用嗎？**
A: 把網址傳給朋友，每個人手機各自安裝即可。目前各自獨立記錄，如需多人共用同一份記錄，需要加後端（可之後升級）。
